What's this about?

I'm a big fan of CSS code golf. I play it here. If I score in the top 10, I'll write about it. In the beginning I'll explain a lot, but won't re-explain a trick that I've already explained. Here are some general tips though:

  • Minify everything
  • Find the shortest possible unit. This means:
    • Dropping the unit altogether for some properties like margin and padding. Eg: margin: 5; instead of margin: 5px;
    • 20px = 5vw
    • 40px = 5ch
    Everything else you should try use percentage or q value usually. Q is a bit less than a px. So 10px = 11Q, and 50px = 54q roughly. You'll have to play with the values.
