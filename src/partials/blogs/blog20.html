Alrighty, this was an interesting challenge. Couldn't make as much use of the html and body elements for this one, as I needed to use on the body element and that would clip the background color.

Two tricks worth noting here:

  1. Using #14C5 instead of #B0C1F0 for the background color. This works because the shorter hex code has transparency, and the color produced by using this is a combination of the default white background and this color sitting on top of it. I used this color mixing tool to determine what the closest matching short hex code would be to the longer one.
  2. Using to size the element, instead of padding, width or height. This worked nicely because all the children of the body element have a border, and the desired size can be achieved by scaling the original 30px border.
